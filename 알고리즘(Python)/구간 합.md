# 파이썬 기준

## 구간 합이란?

- 합 배열을 통해 특정 범위의 합 등을 계산하기 위해 사용하는 특수한 알고리즘
    -**합 배열**: 기존 배열에서 특정 원소 index까지의 누적 합을 더한 배열, 시간 복잡도를 줄이기 위해 사용
    - 합 배열 공식: S[i] = S[i-1] + A[i]
    - 코드: 
    ```
    prefix_sum = [0], temp = 0
    
    for i in list:
        temp += i
        prefix_sum.append(temp) #[0, list[0], list[0]+list[1], ...]

    ```
- 구간합 공식: S[j] - S[i-1]
- 코드:
```
prefix_sum[j] - prefix_sum[i-1] #i를 포함.
```

- 위 코드를 변형하여 n차원의 넓이 혹은 부피를 구하는 문제도 해결 가능[백준 11660번](ㅇㄹㅇㄹㅇㄹㅇㄹ)


- 기타 정보
    - [[0] * 3] -> [[0, 0, 0]]
    - [[0] for _ in range(3)] -> [[0], [0], [0]]
    - [[0] * 3 for _ in range(3)] -> [[0, 0, 0], [0, 0, 0], [0, 0, 0]]
