### React Native 소개
  - 기본 앱 구성요소로 컴파일되는 JS 오픈소스 모바일 프레임워크

#### 생긴 이유
  - Android, IOS를 동시에 개발하기 위해

#### 장점
  - js only로 앱을 만들 수 있음
  - 재사용성 용이(컴포넌트 구조)
  - 개발속도 빠름(빌드 없이 Live Reloading)
  - 오픈소스임

#### 단점
  - 기능적인 한계(RN이 제공하는 기능이 아니면 로컬개발이 필요함)
  - 대형 프로젝트에서 간간히 성능 이슈가 발생함
  - 구조적으로 로컬보다 성능이 딸림(js -> RN을 통해 변환되는 과정 필요)
