자바스크립트의 역사

==============

- 1995년 '넷스케이프 커뮤니케이션즈' 의 **브렌던 아이크**에 의해 개발됨. 

- 개발 목적은 웹에서 동적인 활동을 가능하게 하기 위함.

- Mocha->LiveScript->JavaScript로 이름 변경. 당시 Java가 인기여서 거기에 편승하기 위함.

- 1996년 마이크로소프트가 IE를 통해 JScript를 출시하고, 1997년에는 ECMA에서 자바스크립트의 표준 사양을 발표함.(호환성을 보장하기 위함.)

- 자바스크립트는 꾸준히 발전해왔고, 현대에는 Node.js와 여러 프레임워크/라이브러리(React, Angular, Vue 등등)의 등장으로 웹/앱 개발의 어느 파트에서나 쓰일 수 있는 만능 언어가 됨.
