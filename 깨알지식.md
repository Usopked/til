## 짤막해서 문서를 따로 만들기 애매한 내용 기록

- ### VScode에서 Github 연동법 : Ctrl+Shift+P -> Git:Clone -> 레포지토리 선택

- ### Python에서 map을 바로 반환하면 객체의 메모리 주소가 반환됨. 
- #### A = map(int, input().split()) -> 메모리 주소
- #### A = list(map(int, input().split())) -> input을 공백(split)으로 쪼갠 각각의 int(value)

- ### 주소 읽는법 : https(프로토콜)://github.io(호스트=도메인):8800(포트)/page1(경로)?id=...=...&(쿼리 문자열)
